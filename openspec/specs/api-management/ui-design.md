## 1. 设计概览

### 1.1 设计原则

- **复用优先**：最大程度复用现有 shadcn/ui 组件和业务组件
- **风格一致**：深色主题、紧凑布局、与现有 API 测试模块视觉统一
- **信息层级清晰**：文档 → 端点 → 详情，逐层展开

### 1.2 设计系统

| 项目 | 规范 |
|------|------|
| 组件库 | shadcn/ui (Radix + Tailwind) |
| 主题 | 深色为主，支持浅色切换 |
| 字号 | 标题 14px/16px，正文 12px，辅助 10px |
| 间距 | 基础单位 4px，常用 8/12/16px |
| 圆角 | 小组件 6px，卡片 8px |

### 1.3 可复用组件清单

| 组件 | 路径 | 用途 |
|------|------|------|
| Card | `components/ui/card.tsx` | 文档分组卡片 |
| Button | `components/ui/button.tsx` | 操作按钮 |
| Badge | `components/ui/badge.tsx` | 标签、计数 |
| MethodBadge | `components/ui/method-badge.tsx` | HTTP 方法标识 |
| Sheet | `components/ui/sheet.tsx` | 右侧滑出面板 |
| Dialog | `components/ui/dialog.tsx` | 确认弹窗 |
| Tabs | `components/ui/tabs.tsx` | 导入方式切换 |
| Input/Textarea | `components/ui/input.tsx`, `textarea.tsx` | 表单输入 |
| EmptyState | `components/ui/empty-state.tsx` | 空状态 |
| Checkbox | `components/ui/checkbox.tsx` | 批量选择 |
| Collapsible | `components/ui/collapsible.tsx` | 展开/折叠 |
| Skeleton | `components/ui/skeleton.tsx` | 加载态 |
| Alert | `components/ui/alert.tsx` | 错误提示 |

## 3. API 导入页面

### 3.1 复用现有 ImportApiSheet

导入功能已存在于 `import-api-sheet.tsx`，需做以下调整：

**现有能力（保持）：**
- 三种导入方式：粘贴内容 / URL / 上传文件
- 解析结果展示：端点列表 + 选择导入
- 格式识别：OpenAPI 3.x / Swagger 2.0 / Postman / HAR / cURL

**需增强：**
1. 新增 `documentName` 字段（用户可自定义文档名称）
2. 导入成功后跳转到 API 管理页（而非测试用例页）
3. 支持更新已有文档（触发变更检测）
